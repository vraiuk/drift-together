# «СПЛАВ» — Use Cases

**Версия 0.1** · сценарии взаимодействия игроков с игрой · дополняет ГДД v0.1

## Акторы

- **Хост** — создатель сессии (любой из игроков)
- **Гость** — игрок 2–4
- **Река (система)** — течение, препятствия, физика, спауны
- Принцип: все игроки равноправны, роли возникают ситуативно и ротируются

---

## UC-01. Сбор команды и старт сплава

- **Акторы:** Хост, Гости
- **Предусловия:** игра установлена у всех
- **Основной поток:** Хост создаёт сессию → гости подключаются (invite/код) → выбор «с начала» или «с чекпоинта» → пробуждение на плоту
- **Альтернативы:** старт с любой открытой стоянки; замена игрока между сессиями
- **Результат:** команда на плоту, акт 1

## UC-02. Дежурство на руле (плёс)

- **Акторы:** 1 игрок (дежурный); остальные заняты своим
- **Основной поток:** Дежурный встаёт к рулю → корректирует курс влево/вправо → следит за течением и якорем → объявляет препятствие на горизонте
- **Альтернативы:** передать дежурство другому; никто не правит → плот несёт по воле течения (бампы о берег, пропуск притоков)
- **Результат:** плот движется управляемо, команда свободна для хозяйства

## UC-03. Хозяйство на плоту

- **Акторы:** 1–3 игрока
- **Основной поток:** Выбор активности (готовка на костре / ремонт прочности / крафт / сортировка запасов) → физическое взаимодействие → результат в запас или в плот
- **Альтернативы:** костёр разгорелся слишком сильно — комедийный риск; нехватка ресурсов — форсинг вылазки
- **Результат:** еда, прочность, предметы

## UC-04. Рыбалка

- **Акторы:** 1–2 игрока
- **Основной поток:** Взять удочку → выбрать место (заводь перспективнее) → ловля → улов в запас
- **Альтернативы:** крупная рыба тянет — качание плота, помощь друга; не клюёт — смена места
- **Результат:** еда, возможный мем

## UC-05. Падение за борт / толкание

- **Акторы:** 2+ игрока
- **Основной поток:** Бамп или толчок → игрок в воде → плывёт к плоту → вылезает → статус «мокрый» → сушка у костра
- **Альтернативы:** в пороге — относит течением, команда спасает или ждёт ниже; долго мокрый — мягкий штраф (тёплый/быстрый у костра снимается)
- **Результат:** мем, лёгкие последствия, никакой смерти

## UC-06. Препятствие: стоянка и разведка

- **Акторы:** вся команда
- **Основной поток:** Препятствие на горизонте → решение встать (якорь/стоянка) → разведчик идёт или плывёт вперёд → доклад → план и распределение ролей
- **Альтернативы:** не встали — проход «с листа»; разведчик нашёл обход, волок или ресурсы
- **Результат:** план прохождения

## UC-07. Прохождение порога (all hands)

- **Акторы:** вся команда
- **Основной поток:** Занять посты (левый борт, правый борт, руль, скорость) → старт → кооперативная гребля и команды вслух → выход из порога
- **Альтернативы:** ошибка → переворот (UC-08); кто-то не успел на пост — импровизация
- **Результат:** препятствие пройдено, подсчёт потерь

## UC-08. Переворот плота

- **Акторы:** вся команда
- **Основной поток:** Потеря баланса → плот кувыркается → все в воде → ловля плавающих вещей → совместный переворот обратно → фиксация потерь
- **Альтернативы:** часть вещей уплыла безвозвратно; кто-то отплыл далеко — смешное спасение
- **Результат:** минус ресурсы, плюс сцена; голод форсит рыбалку

## UC-09. Потеря плота

- **Акторы:** вся команда
- **Основной поток:** Якорь не держит / все ушли → плот уплывает → движение вниз по берегу → плот застрял в камнях → возвращение и ремонт
- **Правило системы:** река всегда возвращает плот — зацеп ниже по течению гарантирован, софтлок невозможен
- **Результат:** минус время и ресурсы, мини-приключение

## UC-10. Береговая вылазка (3+1)

- **Акторы:** 3 игрока (берег), 1 (плот)
- **Основной поток:** Стоянка → трое на берег (сбор, лагерь туристов, звери) → дежурный на плоту (якорь, рыбалка, ремонт) → связь по proximity voice, пока рядом → возвращение
- **Альтернативы:** заблудились; зверь — убегать или охотиться; якорь сорвало → UC-09
- **Результат:** ресурсы, риски, сцены

## UC-11. Волок

- **Акторы:** вся команда
- **Основной поток:** Решение обойти препятствие → разведка маршрута → рубка просеки → крепление канатов → совместное таскание → спуск на воду ниже
- **Альтернативы:** передумали на середине — плот на суше, голод тикает; встречное препятствие в лесу
- **Результат:** обход куплен временем и едой

## UC-12. Развилка

- **Акторы:** вся команда
- **Основной поток:** Развилка видна заранее → короткий спор под давлением течения → выбор рукава → прохождение
- **Альтернативы:** разведка со стоянки, если успели встать; нерешительность — течение выбирает само
- **Результат:** маршрут выбран, второй вариант остался тайной

## UC-13. Установка и перестановка модуля

- **Акторы:** 1+ игрок
- **Основной поток:** Есть ресурсы → выбор слота → установка (парус / тент / кузница / мотор...) → изменение центровки → команда адаптируется (крен!)
- **Альтернативы:** демонтаж или замена — спор «что важнее»; перекос → риск UC-08
- **Результат:** плот изменился, выбор необратим до переделки

## UC-14. Стоянка-чекпоинт / завершение сессии

- **Акторы:** вся команда
- **Основной поток:** Стоянка-кемп → костёр, еда, ремонт → автосейв → выход из игры
- **Альтернативы:** продолжение позже той же командой; новый состав с чекпоинта
- **Результат:** прогресс сохранён

## UC-15. Финал: водопад и отчёт о сплаве

- **Акторы:** вся команда
- **Основной поток:** Водопад на горизонте → выбор: тяжёлый волок в обход или риск → финальный сет-пис → пристань, цивилизация → титры + **отчёт о сплаве** (статистика, мемные номинации, время, челлендж-пометки)
- **Альтернативы:** «приплыли на яхте» против «доползли на двери от шкафа» — отчёт отражает состояние плота и команды
- **Результат:** конец игры, шерибельный артефакт

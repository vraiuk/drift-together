# «Сплав» Co-op Slice v0.2 — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 2–4 player online co-op raft run down the existing «Туманный лес» river: join by code, crewed raft with posts, overboard/wet status, campfire checkpoint, finish report with nominations.

**Architecture:** Netcode for GameObjects on Unity Transport. Host-authoritative raft/game state, owner-authoritative player avatars. Two connection paths: Unity Relay join-code (when UGS project is linked) and direct IP (always works, no services). All game objects built in code; the two network prefabs (Raft, PlayerAvatar) are generated as assets by the editor bootstrap and loaded from Resources.

**Tech Stack:** Unity 6000.3.20f1, URP 17.3, NGO 2.x, com.unity.services.{core,authentication,relay}, existing DriftTogether.Runtime code (RiverSpline/RiverFlow/LevelBuilder/HullIntegrity/AudioSynth/UIBuilder/DialogueQueue).

# Drift Together — «Туманный лес» (первая итерация)

Атмосферное аркадное приключение на воде: вы сплавляетесь на байдарке вниз по
туманной лесной реке вместе с пассажиром — саркастичным ожившим тапком по имени
Тапок-Тим. Впереди развилка из двух маршрутов, светящиеся грибы, обязательный
привал у костра и финишная арка. Продолжительность — примерно 8–12 минут.

## Запуск

### Windows
1. Распакуйте `Builds/DriftTogether-Windows-x86_64.zip`.
2. Запустите `DriftTogether.exe`. Установленный Unity не требуется.

### macOS
1. Распакуйте `Builds/DriftTogether-macOS-Universal.zip`.
2. Запустите `DriftTogether.app` (universal: Apple Silicon и Intel).
3. Приложение не подписано и не нотариализовано: при первом запуске macOS может
   показать предупреждение. Откройте через правый клик → «Открыть», либо
   разрешите запуск в «Системные настройки → Конфиденциальность и безопасность».

## Управление

| Действие | Клавиатура | Геймпад |
| --- | --- | --- |
| Грести вперёд | `W` / `↑` | левый стик вверх |
| Тормозить / назад | `S` / `↓` | левый стик вниз |
| Повороты | `A`, `D` / `←`, `→` | левый стик |
| Взаимодействовать | `E` | нижняя кнопка (A / ✕) |
| Вернуться на последнюю точку | `R` | правая кнопка (B / ◯) |
| Пауза | `Esc` | Start |

## Как играть

- У байдарки 3 единицы прочности. Сильные удары о камни и брёвна снимают по
  одной; после удара действует короткая неуязвимость.
- При нулевой прочности вы возвращаетесь на последний checkpoint с полной
  прочностью — постоянного поражения нет.
- По пути можно собрать 5 светящихся грибов (необязательно).
- На развилке выберите маршрут: узкий медленный «Тихий канал» или быстрый
  каменистый «Шумный ручей».
- Остановка у костра обязательна: подплывите к пристани и нажмите `E` — костёр
  чинит байдарку и становится новой контрольной точкой.
- На финише — экран результатов: время, маршрут, грибы, столкновения,
  восстановления. Можно начать заново или выйти в меню.

## Настройки

В главном меню и в паузе: общая громкость и отзывчивость камеры.
Настройки действуют в течение текущего запуска.

## Исходники

Полный Unity-проект — в папке `DriftTogether/` (Unity 6000.3.20f1, URP).
